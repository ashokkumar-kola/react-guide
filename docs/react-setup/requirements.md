# Requirements

## Node.js Setup

- **Node.js** – runtime for React apps
- **npm** – package manager included with Node.js
- **nvm** – manage and switch between Node versions

## IDE / Editors

### Visual Studio Code

A lightweight editor with strong React support.

**Recommended Extensions**

- ES7+ React/Redux/JS Snippets
- Prettier – Code Formatter
- ESLint
- Tailwind CSS IntelliSense (if using Tailwind)
- GitLens
- Auto Rename Tag
- IntelliSense for CSS class names
- Path Autocomplete

### WebStorm

Full-featured IDE with built-in React and JavaScript tools.

### Git

Version control for code, branches, and collaboration.

## Additional Tools

- **Node Dev Tools** – browser DevTools for debugging React
- **pnpm / yarn** – alternative package managers
- **npx** – run CLI tools without global installs
- **Postman / Thunder Client** – API testing
- **Chrome / Edge** browsers – best DevTools support for React
- **VS Code Terminal / PowerShell / zsh** – command-line environment
